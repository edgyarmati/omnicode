## What OmniCode is

OmniCode is an **OpenCode plugin + launcher**.

It is **not**:
- a custom terminal UI
- a fork of OpenCode
- a continuation of the old Omni-Pi standalone shell work

OpenCode owns the UX and runtime. OmniCode owns the **workflow layer**.
